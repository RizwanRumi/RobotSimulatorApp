Instruction for running Robot Simulator Program:

1. Download latest Jdk and install it.

2. Install Visual Studio Code and open it

3. Install Extension Pack for Java from VS Code Extensions

4. Create a folder path C:\Personal_project and unzip RobotSimulatorJava

5. Check the input file path appropriately

6. Run "Program.java" file
